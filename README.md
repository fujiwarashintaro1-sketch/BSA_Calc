# BSA 基準値検索ツール

入力された BSA（体表面積）に基づき、対応する LVDd / LVDs / LADs の基準値を即座に表示するツールです。

## 🚀 使い方
1. ブラウザで `index.html` を開きます。
2. BSA (m²) を入力します（例: 0.85）。
3. 対応する数値が自動的に表示されます。

## 📊 仕様・データについて
- **対応範囲**: BSA 0.01 ～ 1.50
- **検索ロジック**: 入力された数値に最も近い BSA 値のデータを参照します。
- **データ出典**: 九州厚生年金病院と書かれた紙から作成しましたが、出展元は発見できていません。

## ⚠️ 注意事項（必ずお読みください）

### 1. 入力値の制限
本ツールは、BSA 0.01 から 1.50 の範囲にのみ対応しています。範囲外の数値が入力された場合は、結果が表示されません。

### 2. 数値の近似について
本ツールは入力値に最も近い値をリストから検索して表示します。完全一致するデータがない場合は近似値を参照するため、厳密な計算が必要な場合は元データを確認してください。
